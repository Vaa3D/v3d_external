Module v3d

Python module for interacting with the V3D visualization program.

The v3d python module enables control of the V3D biomedical volume visualization program from the python programming language. The v3d module exposes the V3D plug-in API; thus most things that can be done from a V3D plug-in can also be done interactively from the V3D python console.
